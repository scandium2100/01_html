inline 형식 요소

        - 화면을 수평 분할
        (가로화면/영역을 잘라서 열로 분할 (김밥 썰기))

        - width/height 속성 지정 불가능
            -> 작성된 내용 만큼의 크기만 차지함
        
        -inline 형식 요소 종류
            b, i, strong, em, mark, span(영역 분할)
inline 형식은 행을 잘라서 열을 만드는 성질을 가지고 있음 줄이 바뀌지 않고 계속 옆으로 작성된다 (inline형식 요소는 글자 취급)

block vs inline : 영역 지정 방식의 차이

        -block 형식 요소 : 항상 직사각형의 형태로 영역을 지정
        -inline 형식 요소 : 작성된 내용(content) 단위로 영역을 지정
        block : 행 나눔, 크기지정 가능
        inine : 열 나눔, 크기지정 불가
