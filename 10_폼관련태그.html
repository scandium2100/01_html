form(모양, 형식) 태그

        - 입력 양식을 작성하는 판(영역)
        - 내부에 작성된 값을 지정된 경로(html파일, 서버)로 제출하는 역할

        ex) 회원가입, 로그인, 글 작성, 검색 등등..

        [속성]
        1) action : 제출될 파일 또는 서버 경로 작성
        2) method : 제출 방식 지정
        3) name : 이름 지정(java_script에서 사용)
        4) target : 제출된 페이지를 현재 탭/새탭 어디서 볼지 지정

button관련 태그

        - 버튼을 만드는 태그
        - type 속성에 따라 용도가 달라짐
        type="submit" : form태그 내부에 작성된 경우
                        해당 form에 모든 input값을 action 경로로 제출하는 버튼
        type="reset" : form태그 내부에 작성된 경우
                        해당 form에 모든 input값을 제거하는 버튼(초기화)
        type="button" : 아무 기능도 없는 버튼 -> JS를 이용해서 원하는 기능 추가 가능
